接下俩介绍 plugins upload 里面的几个参数

  • serverPath : 文件上传 Controller 的路径，用于后台处理上传图片
  • fileFieldName : 后台接收文件的名字，也就是 <input name='upload'>
  • usage : 是海痴自己加上的，用于判断上传来源，方便管理，这个不是官方参数，需要自己修改 js 源码
